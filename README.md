Project Name: Admin Panel

Description:
This is a web-based Admin Panel used to manage users, orders, and system data efficiently with a responsive interface.

Features:
1. Admin authentication system
2. Dashboard with analytics
3. User management (add, update, delete)
4. Order tracking and management
5. API integration for dynamic data
6. Responsive design

Tech Stack:
Frontend: HTML, CSS, JavaScript, React.js, Bootstrap

Author:
Divya Srivastava

Notes:
This project demonstrates full-stack development using API integration, SQL database management, and responsive UI design.
